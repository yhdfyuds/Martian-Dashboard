<h1 align="center">Martian Dashboard</h1>

## What does this do?
This is a dashboard to allow inhabitants of Mars to check their shuttle statics and health.

## Features 
- Login authentication for all martians
- Dashboard displays stats for martian logged in, such as temperature
- Autocorrects stats for maximum performance
